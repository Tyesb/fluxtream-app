Widgets Guide

We all have different visualization needs

Because owning your data also means having the freedom to visualize it the way you want, Fluxtream can be customized with community contributed dashboard widgets. Provided the right CORS headers configuration, those widgets can be hosted on an external server (e.g. S3). This means that tech-savvy end-users are able the tailor the UX to their own, specific needs without having to host their own server instance.

All your connector data is already there

Widgets are accessible from the dashboards tab in the Calendar app and they readily have access to what we call the "digest" which contains all of the user's connector data for a day, a week or a month
